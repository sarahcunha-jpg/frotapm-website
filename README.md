# Frota PM - Sistema de Gestão da Frota

## Visão Geral

Sistema de gestão de frota para a Polícia Militar com funcionalidades de cadastro de viaturas, controle de manutenção, rastreamento e dashboard de indicadores.

## Estrutura do Projeto

```
frotapm-website/
├── frontend/          # Aplicação front-end (HTML, CSS, JavaScript)
├── backend/           # API back-end (Node.js + Express)
├── database/          # Schema e scripts de banco de dados
└── README.md
```

## Tecnologias

- **Frontend**: HTML5, CSS3, JavaScript, Bootstrap
- **Backend**: Node.js, Express
- **Database**: Supabase / Firebase / MySQL
- **Real-time**: WebSocket / Firebase Realtime / Supabase Realtime

## Como Começar

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
# Abra index.html em um servidor local ou use uma ferramenta como Live Server
```
